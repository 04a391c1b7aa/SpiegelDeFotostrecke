SpiegelDeFotostrecke.user.js
============================

Ein UserScript ( http://userscripts-mirror.org/scripts/show/136377 ) für GreaseMonkey / Tampermonkey um alle Fotos einer Fotostrecke auf spiegel.de auf einer Seite darzustellen.
